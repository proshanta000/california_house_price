California House Price Predictor

Welcome to the California House Price Predictor! This application uses a machine learning model to estimate the median house value for different block groups in California. By analyzing key features of a community, we can provide an accurate prediction to help you understand local real estate values.

Dataset Overview

  • • Source: Derived from the 1990 U.S. Census, with each entry representing a census block group.
  • • Target Variable: Median House Value, expressed in hundreds of thousands of dollars ($100,000). This is the value our model predicts.
  • • Total Instances: 20,640 unique block groups are included in the dataset.

Key Attributes

  • • MedInc: Median income for the block group.
  • • HouseAge: Median house age in the block group.
  • • AveRooms: Average number of rooms per household.
  • • AveBedrms: Average number of bedrooms per household.
  • • Population: The total population of the block group.
  • • AveOccup: Average number of household members.
  • • Latitude: Block group latitude.
  • • Longitude: Block group longitude.
